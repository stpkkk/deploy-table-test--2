<h2>Использовались библиотеки</h2>
<p>UI:</p>
<p>moment - для форматирования даты</p>
<p>sass - препроцессор CSS</p>
<p>Backend:</p>
<p>express -  фреймворк для Node.js</p>
<p>pg - для подключения к базе данных PostgreSQL</p>
<p>cors - для дополнительных заголовков http</p>
</div>
<div>
<h2>О задании</h2>
    <p>Тестовое задание на должность Frontend-разработчика в компании Welbex с использование фреймворка React.js, PostgreSQL, Sass. Реализовано одностраничное веб-приложение для отображения таблицы с возможностью сортировки и пагинации. Backend часть реализована в виде СУБД PostgreSQL и приложения pgAdmin.</p>
